#### Aşağıdaki kodda yapılan işlev çağrılarının durumunu belirtin. Geçerli mi değil mi? Geçerli ise çağrılan işlev hangisidir?

```
void func(void *);  //1
void func(bool); //2

void foo()
{
	double x = 1.0;

	func(0);
	func(nullptr);
	func(&x);
	func(x);
}
```
